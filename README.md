![us1](https://github.com/user-attachments/assets/f7534d5a-1b78-4566-8e98-918d7acf597b)#  A.H. Electricity- Utility Bill Calculator

A full-stack web app for calculating electricity utility bills with admin-configurable rates and bills can be printed as PDF.

## Live Demo:
- Frontend (Actual Implementation): [https://ah-utility-bill-calculator.vercel.app/](https://ah-utility-bill-calculator.vercel.app/)
- Backend (API Endpoint Logic): [https://acme-utility-bill-calculator-backend.onrender.com/](https://acme-utility-bill-calculator-backend.onrender.com/)


## Features
- **Admin Panel:** Update rates, VAT, and service charge (PIN protected)
- **User Calculator:** Input units, get instant bill breakdown
- **PDF Export:** Download results as PDF
- **Live Config:** Real-time rate updates

## Tech Stack
- **Frontend:** React (TypeScript)
- **Backend:** NestJS (TypeScript)
- **Database:** PostgreSQL (Supabase)
- **Deployment:** Vercel (Frontend) and Render(Backend)
